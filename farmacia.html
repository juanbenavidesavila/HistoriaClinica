Solección de aréas


Aquí comienza el ingreso de datos de la historia clinica, al rellenar el formulario se enviaran los datos a un email en contabilidad para emitir la factura directamente en el diam y uno a administración para subir la atención médica del pasciente a su historia clinica en su base de datos:
Intrucciones:
  • 1. La primera seccion de "Datos Personales", es un CRUD hacia la base de datos global, el campo del RUN (Número unico de identidad) del paciente hace mach con el ID de la base de datos global, y al hacer clic en botón "Consultar" entrega el historial clinico ya subido anteriormente.

  • 2. Al mismo tiempo, el primer campo ingresa la información a una base de datos local del propio hospital con el botón "Ingresar", quien luego de la atención, la información se envia al departamento de informatica, para subir la consulta con todos los detalles a la base global.

  • 3. El campo de datos primario, es el primer control de enfermería, peso, IMC, rito cardiaco, temperatura. la cual hacer clic en el Botón "Ingresar" actualiza los campos de la abse de datos local.

  • 4. Luego el médico quien atiende al pasiente, ingresa los datos de la atención y diagnostico con el botoón "Ingresar" a la base de datos local.

  • 5. A su vez, una vez atendido se rellena la sección de factura, donde se agregan los datos de los costos de atención a traves del botón "Ingresar".

  • 6. Ahora, con toda la información de atención, todas las secciones estan en una base de datos global, al rellenar cada una de la sección debe llegar una notificación al departamento de informatica o administración, quien debe consultar con el RUN la información antes ingresada para realizar la factura correspondiente y subir la atencion a su historia clinica global.
